# Basic_chatbot

A basic rule‑based Python chatbot that responds to:
- `hello` → “Hi!”
- `how are you` → “I’m fine, thanks!”
- `bye` → “Goodbye!”
- Any other input → “Sorry, I don’t understand that.”
